# Apple Stock Price Analysis & 2027 Forecast

## Overview
Analysis of Apple (AAPL) stock data and prediction of 2027 stock prices using Python.

## Tools
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Work Done
- Data cleaning
- Data visualization
- Model building (Linear Regression)
- Price prediction for 2027

## Dataset
The dataset contains historical stock price data of Apple (AAPL) including Date, Open, High, Low, Close, Adjusted Close, and Volume.

## Results
- The model shows an overall upward trend in Apple stock prices.
- Predicted prices for 2027 indicate continued growth based on historical patterns.

## How to Run
1. Open the Jupyter Notebook
2. Run all cells step by step
3. View visualization and prediction output
